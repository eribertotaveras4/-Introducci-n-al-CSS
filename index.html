DISEÑADOR DE PÁGINAS WEB, CSS Y JAVASCRIPT
ACTIVIDAD: 1

  1. Qué es CSS?

    2. CSS: (Hojas de Estilo en Cascada) es un lenguaje que se utiliza para describir la presentación de un documento HTML, especificando cómo deben aparecer los elementos en la pantalla.

  2. Cuál es la función principal de CSS?

    3. La función principal de CSS es separar el contenido de la presentación, lo que permite definir estilos como colores, fuentes y espaciados de una manera más eficiente.

  3. Cómo se puede incluir CSS en un documento HTML?

    • CSS se puede incluir en un documento HTML de tres maneras:


    • Inline: Directamente en la etiqueta HTML usando el atributo style.

    • Interno: Dentro de una etiqueta style en la sección del HTML.

    • Externo: En un archivo css separado, enlazado al HTML con una etiqueta link

  4. Qué son los selectores en CSS?

    5. Los selectores en CSS son patrones que seleccionan los elementos HTML a los que se les aplicarán los estilos. Pueden ser simples, como etiquetas (h1, p), clases (.clase), y IDs (#id)

  5. Qué son las propiedades y valores en CSS?

    6. Propiedades y valores: En CSS, las propiedades son los aspectos del elemento que se desean cambiar, como el color o el tamaño de la fuente.

  6. Cómo se aplica el modelo de caja en CSS?

    7. El modelo de caja en CSS trata a cada elemento como una caja con cuatro partes: contenido, padding (relleno), borde y márgenes. Esto ayuda a diseñar y espaciar los elementos de la página.

  7. Qué es el "cascading" en CSS?

    8. Cascading se refiere a cómo las reglas CSS se aplican de forma jerárquica, desde las más generales a las más específicas, permitiendo que los estilos se combinen y se sobrescriban según las necesidades.

  8. Qué son las media queries?

    9. Las media queries son reglas CSS que permiten aplicar estilos diferentes según las características del dispositivo, como el tamaño de la pantalla. Son fundamentales para crear diseños responsivos.

  9. Cuáles son algunos estilos comunes que se pueden aplicar?

    10. Algunos estilos comunes en CSS incluyen cambiar el color de fondo (background-color), el color del texto (color), la fuente (font-family), el margen (margin), el padding (padding) y la alineación del texto (text-align).

  10. Cómo se puede usar CSS para crear un diseño responsivo?

    11. El diseño responsivo en CSS se logra usando unidades relativas (%, em, rem), media queries para ajustar los estilos según el tamaño de la pantalla, y técnicas como flexbox y grid para distribuir los elementos de manera flexible y adaptable.
